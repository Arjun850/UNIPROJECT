# 🎓 UNIPROJECT

A web-based University Project Management System that connects students and faculty on a single platform. Students can explore available projects, apply for them, and track their applications, while faculty members can create projects, manage applications, and publish project blogs.

---

## 🚀 Features

### 👨‍🎓 Student Module
- Student Authentication
- Student Dashboard
- Edit Profile
- Browse Available Projects
- Apply for Projects
- View Application Status
- View Project Blogs

### 👨‍🏫 Faculty Module
- Faculty Authentication
- Faculty Dashboard
- Create New Projects
- Manage Student Applications
- Publish Project Blogs
- Update Project Details

### 🔐 Authentication
- Secure Login System
- Supabase Authentication
- Session Management

---

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Arjun850/UNIPROJECT.git
```

### Navigate to the Project

```bash
cd UNIPROJECT
```

### Install Dependencies

```bash
npm install
```

### Run the Project

Open `index.html`

or start your local development server if required.
